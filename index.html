快速入门

统一

不同项目使用一致的开发, 构建及发布流程. 在开发中强制使用 ESLint 作为格式检查器, 使代码风格具有高度一致性.

复用

无论是 HTML 还是图表, 都进行组件化封装, 达到组件的高度可复用性.

效率

基于 React + Redux 构建, 借用大量开源社区成熟框架及组件, 满足绝大部分开发时的组件和功能需求. 使用 Webpack 进行自动化构建, 简化以往繁琐的构建流程.
