组件化 UI

丰富, 灵活, 实用的基础组件, 满足各种业务需求. 可以快速构建出界面美观, 体验一致的大型应用.

可控的 State

使用 Redux 对应用程序的状态变化进行管理. 配合 DevTools 您可以浏览并回放 state 的变化过程.

即时反馈

当您保存修改后的 JS 和 CSS 文件后, 浏览器立即自动热加载, 不再需手动刷新. 即使修改了底层代码, 更新后所有状态也都将得以保持.

自动化

提供命令行接口工具, 使您可以一键启动调试, 运行测试用例和构建发布版本, 不需再进行任何构建工具的配置工作.

下一代 JavaScript

支持使用最新 ECMAScript 规范版本进行应用的构建. 即从现在起就可使用箭头函数, 解构赋值, 修饰器等特性进行开发.

下一代 CSS

基于模块化的 CSS 解决方案, 在大型多人项目中, 再也不用担心命名冲突和样式覆盖. 基于组件化的加载方式带来更高的性能.

跨平台

支持使用一致的方式进行 web 端, 移动端和桌面端的开发和构建. 不同平台代码具备高可复用性.

原生应用

移动端使用 react-native 进行构建, 封装大量常用移动组件, 使用最小的学习成本开发原生移动应用.

基于模版创建

使用命令行即可初始化项目模版, 是您可以快速的开始构建您的应用.
