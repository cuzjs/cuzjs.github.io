NumberInput

计数器 NumberInput #

使用 NumberInput 标签，可以快速生成一个简单的计数器。通过 value 属性可以定义计数器的初始值。 step 属性可以定义每次点击数值的改变量，默认为1。而使用 max 和 min 属性则可以定义计数器计数的上限值和下限值。
