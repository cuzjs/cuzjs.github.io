CheckBox

多选按钮 #

对 <input /> 使用 type="checkbox" 可以生成多选按钮，添加 checked 属性会使按钮默认被选中。

多选按钮组 #

将 <Checkbox> </Checkbox> 和 <CheckboxGroup /> 组合使用,可以生成一个多选按钮组.
