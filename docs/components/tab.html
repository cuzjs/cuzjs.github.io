标签页

一个基本的标签页 #

通过 <Tab /> 标签，可以快速的创建出一个标签页。将多个标签页放在 <Tabs /> 中，可以生成一组标签。

胶囊式标签页 #

HTML 标记相同，但在 <Tab /> 中使用 pills 类。

Props #

Tabs #

名称 类型 缺省值 描述
defaultActiveKey any 设定默认的处于 active 状态的 Tab
bsStyle 以下之一："tabs","pills" "tabs" tabs 导航栏的样式。
id string 或 number
position 以下之一："top","left","right" "top"
