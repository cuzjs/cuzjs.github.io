Pagination #

使用 <Pagination /> 标签可以为你的页面快速生成一个简洁的分页。

属性选项 #

items={number} 定义了分页的数量。pre 和 next 属性可以给分页生成向左和向右的箭头按钮。 当页数过多的时候，添加 ellipsis 则会以 ... 代替部分页码以保持组件清爽。 maxButtons={number} 定义显示最大的页码按钮数。 activePage={number} 表示当页被选择的页码。

PageSize #

定义每页显示多少条数据。

PageTo #

定义跳转到分页中的具体某个页面。

Props #

名称 类型 缺省值 描述
activePage number 1 处于 active 状态的页码序号
ellipsis boolean 或 node true 当值为 true 的时候，会将过长页码以("...")的形式显示
first boolean 或 node false 当值为 true 的时候，会显示("<<")的页码元素
last boolean或 node false 当值为 true 的时候，会显示(">>")的页码元素
items number 1
maxButtons boolean 0
next boolean 或 node false 当值为 true 的时候，会显示(">")的页码元素
prev boolean 或 node false 当值为 true 的时候，会显示("<")的页码元素
onSelect function
