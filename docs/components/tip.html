提示

Tooltips #

使用 <Tooltips /> 标签可以快速生成一个小巧的提示。

配合 OverlayTrigger 一起使用 #

将 Tooptips 配合 OverlayTrigger一起使用，生成针对某个组件的提示。

Props #

Tooltip #

名称 类型 缺省值 描述
placement 以下之一："top","right","left","bottom" "right" 设置 Tooltip 的位置方向
id (required) string 为了易于使用，必须添加该属性 @type{string} @required
positionLeft number 距离左侧的距离值
positionTop number 距离上方的距离值
title node 标题文本

OverlayTrigger #

名称 类型 缺省值 描述
overlay (required) node 在目标元素旁边显示的一个元素或者文本

组件参考 #

React-Bootstrap官方文档: http://react-bootstrap.github.io/components.html#tooltips
