代码框 #

使用 <Code /> 标签可以生成一个代码框.

Props #

名称 类型 缺省值 描述
value string 代码框内展示的代码
options Object 代码框的参数配置
lineNumbers boolean false 是否显示行号
readOnly boolean false 是否可编辑
mode 以下类型之一:'javascript','CSS',more... 展示代码类型
