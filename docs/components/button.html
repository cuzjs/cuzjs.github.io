一个 Button #

这里是一个button的示例,你可以修改代码查看不同效果

不同样式的 Button #

通过在bsStyle属性中添加下列类，可以快速生成带有预定义样式的Button。

不同尺寸的 Button #

按钮有正常和small两种尺寸。默认为正常尺寸，通过添加bsSize="small"可以选择小尺寸按钮。

Button的活动状态 #

active 属性可以让表格处于活动状态。

Button的禁止状态 #

按钮的禁止状态会使点击等事件无效化，通过添加disabled可以实现禁用按钮。

按钮组Button Grounps #

将将<Button />放入<ButtonGroup />中可以实现按钮组。

ButtonGroup #

名称 类型 缺省值 描述
block boolean false 使按钮门成为块级元素，仅仅当和 'vertical' 一同使用的时候有效。
bsClass string "btn-group" 基本的CSS类名和前缀组件。一般来说，当体统新的样式给组件时，应该只改变bsClass。

vertical boolean false
