Box

带跳转的 Box #

使用 <Box /> 标签可以快速生成一个 Box 。添加 viewMore 和 viewMoreText 属性可以生成一个带跳转链接的 Box。

可隐藏和显示的 Box #

添加 showMoreText 和 hideText 属性，可以生成一个向下显示或隐藏的 Box。
