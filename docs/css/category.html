类目选择

Category #

使用 <Category /> 标签可以快速生成一个类目选择的组件.data 属性接受一个 json 格式的值,用于生成类目的级别和内容.

ButtonGroup #

名称 类型 缺省值 描述
confirmButton boolean false 类目选择中是否显示 '确认'的按钮
cancelButton boolean false 类目选择中是否显示 '取消'的按钮
data json 用于生成类目内容和级别的数据源
onSelect function 确认选中时的事件
