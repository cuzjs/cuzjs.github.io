Table #

带排序接口的 Table。

Props #

Table #

名称 类型 缺省值 描述
width (required) number 表格的宽度(px)
height number 表格的高度(px)
maxHeight number 表格的最大高度。height 和 maxHeight两者必须要有一个存在
rowsCount (required) number 表格的行数
rowHeight number 表格每行的高度(px)
defaultSortColumn string 表格初始化时，使用那一列的关键字进行排序
defaultSortType 以下之一："asc"、"desc" "desc" 初始化时排序的方式
onSortChange function 点击排序按钮的时候触发的事件
overflowX 以下之一："hidden"、"auto"
overflowY 以下之一："hidden"、"auto"
headerHeight (required) number 0 表头行的高度(px)
footerHeight number 0 表格 footer 的高度(px)

Column #

名称 类型 缺省值 描述
columnKey string 或 number 该列的关键词
align 以下之一："left"、center、right "left" 对齐方式
fixed boolean false
header node 或 function 存放该列表头元素
cell node 或 function 该列表格主体部分元素
footer node 或 function 该列 footer 部分元素
columnKey string 或 function 表示该列得关键字
width (required) number 该列的宽度(px)
minWidth (required) number 该列的最小宽度(px)
maxWidth number 该列得最大宽度(px)

Cell #

名称 类型 缺省值 描述
height number 该 cell 的高度
width number 该 cell 的宽度
