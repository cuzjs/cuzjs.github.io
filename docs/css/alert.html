Alert

Alert #

使用 <Alert /> 标签可以创建一个弹出框，对 bsStyle 属性添加 sucess 、 info 、 warning 、 danger 这些类可以改变 Alert 的样式。

大尺寸的 Alert #

对 <Alert /> 标签的 bsSize 属性添加 large 类可以生成大尺寸的 Alert 。

Alert #
