状态管理

cuz 使用 Redux 对应用程序的状态进行管理.

Redux #

如果您还没有使用过 Redux, 强烈建议您首先阅读它的官方文档 (中文翻译).

Redux 是 JavaScript 状态容器, 提供可预测化的状态管理.

可以让你构建一致化的应用, 运行于不同的环境(客户端, 移动端和服务器), 并且易于测试.
