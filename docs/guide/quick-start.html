快速开始

Step 1. 命令行工具 #

cuz 的运行依赖于 NodeJS 0.12.x 及以上(< 5.0.0), 运行以下命令安装命令行工具

$ npm install cuz-cli -g

Step 2. 创建项目 #

这一步将创建文件夹, 并安装本地 cuz

$ mkdir cuz-project && cd cuz-project
$ npm install cuz

Step 3. 初始化项目 #

这一步将会使用模版文件初始化一个示例项目

$ cuz init

Step 4. 启动调试 #

一键启动调试，访问 http://127.0.0.1:5000 查看效果

$ cuz
