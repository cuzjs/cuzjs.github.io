重置样式

sanitize.css #

cuz 采用 sanitize.css 对浏览器的样式重置, 使浏览器渲染更符合开发者的预期, 并统一不同浏览器的默认表单组件样式不一致的问题.

更早时期的选择是 normalize.css 或 reset.css, 和它们比起来 sanitize.css 具有更好的灵活性并且兼容下一代 CSS 的特性(比如 CSS variables).

请查看官方文档以了解更多.
