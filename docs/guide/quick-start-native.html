快速开始 (React-Native)

Step 1. 安装命令行工具 #

cuz-native 的运行依赖于 NodeJS 4.0.0 及以上(< 5.0.0), NodeJS 安装完成后执行以下命令

$ npm install cuz-cli -g

Step 2. 创建项目文件夹, 安装 cuz 核心依赖 #

使用命令行进行安装

$ mkdir native-project && cd native-project
$ npm install cuz-native

Step 3. 初始化项目 #

使用命令行进行初始化并允许 node_modules 补丁

$ cuz init
$ node patch.js

Step 4. 启动调试 #

在 Xcode 中打开 ios/App.xcodeproj 并且点击运行 (CMD + R)。
